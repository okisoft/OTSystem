# OTSystem

### 初期設定

* `ruby -v`でruby のバージョンが`2.4.3`であるかを確認する

* bundler のインストール

```
$ gem install bundler
```

* Gemfileがあるレポジトリで以下を実行

```
$ bundle install --path vendor/bundle
```

* 正しく動いているかの確認

```
$ bundle exec rails s
```
